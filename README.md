Данный бот создан для отслеживания заявок с определенным статусом.

Ссылка на боты:
- @Warning0Bot

---

Бот подключается к DWH, создает в нем свою таблицу, получает данные из необходимых таблиц и, после их анализа, вносит в свою таблицу.
